# Stock Portfolio Tracker

## Description
A Python console application that helps users manage a stock portfolio.

## Features
- Add stock symbols
- Calculate total portfolio value
- Display portfolio summary
- Save results as TXT or CSV

- CSV Module
- Datetime Module
